Introduction

Welcome to the official GRAVE Bot documentation. GRAVE is a powerful, all-in-one Discord bot designed to help you manage, grow, and engage your server community.

This documentation covers everything from initial setup to advanced configuration options. If you need additional help, join our support server.

GRAVE requires Administrator or specific permissions depending on what features you use.

Invite the Bot

To add GRAVE to your server, click the button below or use the invite link:

Add GRAVE to Your Server

Make sure you select the correct server and grant the requested permissions for full functionality.

Permissions

GRAVE needs the following permissions to function correctly:

  • Manage Messages — Required for moderation and purge commands
  • Kick / Ban Members — Required for moderation actions
  • Manage Roles — Required for role assignment features
  • Connect / Speak — Required for music playback
  • Send Messages / Embed Links — Required for all responses

Prefix Setup

GRAVE uses slash commands by default. All commands start with /. You can also configure a custom text prefix using:

/config prefix [your-prefix]

Logging

Set up a log channel to track all moderation actions and bot events:

/config logs #channel

The log channel will receive messages for bans, kicks, warns, role changes, and more.

Auto Roles

Automatically assign roles when new members join your server:

/autorole set @role
/autorole remove @role

Moderation Module

GRAVE's moderation module provides a comprehensive set of tools for keeping your server safe.

  • Ban, kick, mute, and warn members
  • Auto-moderation with spam, link, and word filters
  • Detailed audit logs for all actions
  • Temporary mutes and bans with automatic expiry

Music Module

Stream high-quality audio directly in voice channels. GRAVE supports YouTube, Spotify, and SoundCloud.

/play [song name or URL]
/queue
/skip

Utility Module

Useful server and user tools available to all members:

  • /userinfo — View info about a member
  • /serverinfo — View server statistics
  • /avatar @user — Get a user's avatar
  • /ping — Check bot latency
